# Medium-Style Blogger (Blogspot) Template

A clean, minimalist, and responsive Blogger (Blogspot) template inspired by the elegant design aesthetics of Medium. This template is perfect for bloggers who value readability, simplicity, and a professional look for their personal or professional blogs.

## Key Features
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Minimalist Aesthetics**: Emphasizes clean typography, balanced spacing, and intuitive navigation.
- **Hero Section**: Includes a customizable header with support for a background image and engaging introductory text.
- **Elegant Sidebar**: Pre-designed sections for "About Me," "Archives," and social media links, all easily customizable.
- **Social Media Integration**: Seamlessly link to your profiles on platforms like GitHub, LinkedIn, Kaggle, Tableau, Blogger, and personal websites.
- **Modern Footer**: A sleek footer displaying your blog title and a customizable attribution section.

## How to Install
1. **Download the Template**:
   - Clone this repository or download the `medium-style-blogger-template.html` file.

2. **Upload to Blogger**:
   - Log in to your Blogger dashboard.
   - Navigate to **Theme** > **Backup/Restore** > **Restore**.
   - Upload the `medium-style-blogger-template.html` file.

3. **Customize the Template**:
   - Set your blog title, description, and content using the Blogger interface.
   - Replace placeholder links in the header, sidebar, and footer with your social media profiles.

4. **Style Adjustments (Optional)**:
   - Customize colors, typography, and layout in the `<b:skin>` section of the HTML to align with your brand.
